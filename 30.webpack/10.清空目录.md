# 清空目录

```js
'use strict'

const path = require("path");
const CleanWebpackPlugin = require("clean-webpack-plugin")

module.exports = {
    output:{
        path:path.join(__dirname,'dist'),
        filename:'[name]_[chunkhash:8].js'
    },
    plugins: [
        new CleanWebpackPlugin()
    ]
};
```

> 每次构建完成之前自动清空出口目录
