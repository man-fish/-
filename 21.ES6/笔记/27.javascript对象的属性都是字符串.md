# javascript对象的属性都是字符串

### javascript对象的属性是字符串

```js
{
	0:"value",
	op:"value"
}
等价于
{
	”0“:"value",
	“op”:"value"
}
```

```js
{
	-1：“value"
}
//报错
{
	”-1“：“value"
}
```

如果属性名是一个表达式则要这么写：

```js
const attr = {
    key:"name"
}

{
	[attr.key]：“value",
	[1+1]:"value"
}
等价于
{
	"name"："value",
	"2":"value"
}
```
