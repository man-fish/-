# 判断数据的存再性+script作用域

1.判断数据是否被定义

```javascript
<script>
console.log(typeof a)
  if(typeof a ==  'undefined'){
    if(typeof b == 'undefined'){
      console.log(1);
    }
  }
</script>
```

> #### 不使用 typeof 会报错。

2.判断数组键是否存再：

```javascript
    var json = {
      "name" : "peter",
      "age" : 6
    }
    console.log(json.length);  
    //undefined
    console.log(Object.keys(json).length)
    //2
```
