# transform变形

## transform

> 这个属性改变不会影响和模型，不会造成dom重排。

#### 属性

##### rotate ( xdeg )

> 旋转（角度）
>
> ```css
> transform: rotate( 0deg )；
> ```

##### scale ( x,y )

> 缩放（x轴,y轴）
>
> ```css
> transform: scale( 1，1 );
> ```

##### translate ( x,y )

> 平移（距x,距y）
>
> ```css
> transform: translate( x,y )
> ```

##### shew (  x deg,y deg )

> 倾斜（角度）
>
> ```css
> transform: skew(0deg,0deg);
> ```

## 先后顺序

先旋转后缩放

```css
transform: rotate( 0deg )，scale( 1，1 );
```

先缩放后旋转

```css
transform: scale( 1，1 )，rotate( 0deg );
```
