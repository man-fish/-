#### 创建计算字段—其实就是操作查询结果

**拼接字段**   既然查询结果是一对key value 就需要一个别名 ”as“

```
select Concat(name1,' (',name2,')') as newTitle from tableName
```

 **算数计算  +，-，*，/**   既然查询结果是一对key value 就需要一个别名 ”as“

```
select rownam1（*，-，、+）rowname2 as newTitle from tableName 
```
