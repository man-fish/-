# 删除数据

#### 基础写法

```
delete from tablename1 
where rowname1 = "jack"
```

> 要知道删除数据不是删除表，删除的话只能删除一整行，或者删除很多行。

#### 删除列

```
update tablename1 
set rowname1 = null
where rowname1 = "JA"
```

> 删除删除的只是数据不能改变表的结构。

#### ”删除“ 表的数据

```
truncate tablename1
```

这样写的效率要远高于***delete from tablename1***，其实现原理是删除表然后创建一个同名的表

#### 没有重来的机会

不论是**更新**还是**删除**的操作都是无法撤销的。
