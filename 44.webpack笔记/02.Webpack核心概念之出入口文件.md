## 代码

```js
'use strict'

const path = require("path");

module.exports = {
    entry:{
        search:"./src/search.js",
        index:"./src/index.js"
    },
    output:{
        path:path.join(__dirname,'dist'),
        filename:'[name].js'
    },
    mode:"production"
};
```

> 多对象时，entry对象的键对应着output的占位符。
