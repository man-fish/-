# 常量定义🚹

> 常量必须被赋初值，但是变量不需要。

#### 包内常量

```go
package main

import "fmt"
const a = "string"

func const1() {
	fmt.Println(c))
}

func main() {
	const1()
}
```

#### 函数内常量

```go
package main

import "fmt"

func const1() {
	const (
		a    = 1
		b, c = 1, "txt"
	)
	const const2 = "chifan"
	fmt.Println(a, b, c)
}

func main() {
	const1()
}
```
