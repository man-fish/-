

# 美团项目之字母列表

首先，分析一下怎么实现，两个模块之间的对应既然是一对一，那么肯定是用锚点。

### tamplate

![锚点列表](F:\我的笔记\image\锚点列表.png)

> 由图，两个模块都是是**dl>dt+dd*n**，通过锚点产生关联。

```vue
<template>
  <div class="">
    <dl class="m-categroy">
      <dt>按拼音首字母选择：</dt>
      <dd
        v-for="item in list"
        :key="item">
        <a :href="'#city-'+item">{{ item }}</a>
      </dd>
    </dl>
    <dl
      v-for="item in block"
      :key="item.title"
      class="m-categroy-section">
      <dt :id="'city-'+item.title">{{ item.title }}</dt>
      <dd>
        <span
          v-for="c in item.city"
          :key="c">{{ c }}</span>
      </dd>
    </dl>
  </div>
</template>
```

### data数据结构设计

```js
export default {
	data(){
		return {
			block:[{
				title:"A",
				city:['鞍山','安庆']
			},{
				title:"B",
				city:['北京']
			}]
		}
	}
}
```

### script

```vue
<script>
import pyjs from 'js-pinyin'
import axios from 'axios'
export default {
  data(){
    return {
      list:'ABCDEFGHIJKLMNOPQRSTUVWXYZ'.split(''),
      block:[]
    }
  },
  async mounted(){
    let self=this;
    let blocks=[]
    let {status,data:{city}}=await axios.get('http://127.0.0.1:3000/geo/city');
    if(status===200){
      let p
      let c
      let d={}
      city.forEach(item=>{
        p=pyjs.getFullChars(item.name).toLocaleLowerCase().slice(0,1)//b
          //首先我们获取完数据之后根据其首字母拼音来分类，我们用一个拼音库获取其首字母拼音。
        c=p.charCodeAt(0)//b
        if(c>96&&c<123){//二次判断
          if(!d[p]){
            d[p]=[]
          }
          d[p].push(item.name)//["北京",'北平]
        }
      })
      /**此时的数据结构：
      		d:{
      			A:['鞍山'],
      			B:['北京']
      		}
      		Object.entries(d) == 
      			[['A',['鞍山']],['B',['北京']]]
      		   k     v
      */
      for(let [k,v] of Object.entries(d)){
        blocks.push({
          title:k.toUpperCase(),
          city:v
        })
      }
      blocks.sort((a,b)=>a.title.charCodeAt(0)-b.title.charCodeAt(0))
        //记住这种排序方式。
      self.block=blocks
    }
  }
}
</script>

```
