# 解除端口占用

#### 基础命令

```
PID	process-id
在开始-运行-cmd,输入：
netstat –ano 可以查看所有进程

查看 运行 固定端口 应用程序 例如 8080
netstat -ano|findstr "8080"

查看 运行 8080 端口的 应用程序 
tasklist|findstr "1176"(1176 为8080 端口的进程号)

使用命令杀死进程 taskkill /PID 进程号 /F  例：
taskkill /PID /F 1111

查看当前http服务状态。
netsh http show servicestate
```
