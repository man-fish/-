Merge two types into a new type. Keys of the second type overrides keys of the first type.

### Solution

和 `AppendToObject` 的原理基本一致：

```ts
export type Merge<T, U> = {
    [K in keyof T | keyof U]: K extends keyof U
        ? U[K]
        : K extends keyof T
        ? T[K]
        : never;
};
```
